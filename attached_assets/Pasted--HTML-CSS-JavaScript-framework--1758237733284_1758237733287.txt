ช่วยสร้างเว็บไซต์นิทรรศการศิลปะออนไลน์ โดยใช้ HTML, CSS และ JavaScript (ไม่ใช้ framework ภายนอก)  

### โครงสร้างเว็บไซต์:
- index.html → หน้าแรก มีชื่อเว็บไซต์ เมนูนำทาง และสไลด์โชว์ (carousel) ผลงานเด่น
- about.html → หน้าเกี่ยวกับนิทรรศการ
- gallery.html → หน้ารวมผลงาน พร้อมระบบ Filter เลือกหมวดหมู่
- artwork.html → หน้ารายละเอียดผลงาน
- style.css → ไฟล์ตกแต่ง
- script.js → ไฟล์ JavaScript สำหรับสไลด์โชว์และการจัดการข้อมูลผลงาน

### หมวดหมู่ผลงาน:
1. Cubism Art  
   - รายการผลงานแสดง: ชื่อผลงาน, แนวคิด, ศิลปิน  
2. Pop up  
   - รายการผลงานแสดง: ชื่อผลงาน, แนวคิด, ศิลปิน  
3. Pop Art  
   - รายการผลงานแสดง: ชื่อผลงาน, แนวคิด, ศิลปิน  
4. Brochure  
   - รายการผลงานแสดง: ชื่อผลงาน, แนวคิด, ศิลปิน  
5. Sculpture  
   - รายการผลงานแสดง: ชื่อผลงาน, ชื่อกลุ่ม, รายชื่อสมาชิก, แนวคิด  

### หน้าแกลเลอรี (gallery.html):
- ส่วนบนมี Filter (dropdown หรือปุ่ม) ให้เลือกหมวดหมู่:
  - “ทั้งหมด”
  - “Cubism Art”
  - “Pop up”
  - “Pop Art”
  - “Brochure”
  - “Sculpture”
- เมื่อเลือกหมวด จะกรองและแสดงเฉพาะการ์ดผลงานในหมวดนั้น
- ถ้าเลือก “ทั้งหมด” ให้แสดงผลงานจากทุกหมวด
- การ์ดผลงาน:
  - 4 หมวดแรก: thumbnail, ชื่อผลงาน, ศิลปิน, ปุ่มไปดูรายละเอียด
  - Sculpture: thumbnail, ชื่อผลงาน, ชื่อกลุ่ม, ปุ่มไปดูรายละเอียด
- ใช้ grid layout แบบ responsive

### หน้ารายละเอียดผลงาน (artwork.html):
- ถ้าเป็น Cubism Art / Pop up / Pop Art / Brochure:
  - แสดงรูปใหญ่, ชื่อผลงาน, ศิลปิน, แนวคิด
- ถ้าเป็น Sculpture:
  - แสดงรูปใหญ่, ชื่อผลงาน, ชื่อกลุ่ม, รายชื่อสมาชิก, แนวคิด
- ข้อมูลดึงจาก query string (?category=...&id=...)

### หน้าแรก (index.html):
- มีสไลด์โชว์ 5 ผลงานเด่น
- เปลี่ยนภาพทุก 3 วินาทีอัตโนมัติ
- แต่ละสไลด์แสดง: ภาพใหญ่, ชื่อผลงาน, ศิลปิน/กลุ่ม, ปุ่ม “ดูรายละเอียด” ลิงก์ไป artwork.html?id=...

### CSS (style.css):
- ฟอนต์ภาษาไทย เช่น “Prompt” หรือ “Sarabun”
- โทนสี: พื้นหลังเข้ม ตัวอักษรขาว ใช้สีทองเป็น accent
- Header/Nav: พื้นหลังเข้ม, ตัวอักษรขาว, hover สีทอง
- Gallery: grid layout, spacing, hover effect
- Card: border-radius, box-shadow, hover animation
- ปุ่ม: พื้นหลังทอง, ตัวอักษรขาว, hover effect
- Footer: พื้นหลังเข้ม, ตัวอักษรอ่อน, จัดกึ่งกลาง
- Responsive รองรับทั้ง desktop และ mobile

### JavaScript (script.js):
- เก็บข้อมูลผลงานทั้งหมดใน object `galleryData`
- Render ข้อมูลลง gallery.html โดยมีระบบ Filter
- ฟังก์ชัน Filter: เปลี่ยนการแสดงผลเมื่อเลือกหมวดหมู่
- หน้า artwork.html: ดึง id และ category จาก query string เพื่อแสดงรายละเอียด
- index.html: ทำสไลด์โชว์อัตโนมัติเปลี่ยนทุก 3 วินาที
- มี comment อธิบายโค้ด

### ข้อกำหนดทั่วไป:
- เมนูนำทางทุกหน้า: “หน้าแรก”, “เกี่ยวกับ”, “แกลเลอรี”
- ใช้ semantic HTML5 (header, nav, main, footer)
- เนื้อหาทั้งหมดเป็นภาษาไทย
- โค้ดต้องรันได้ทันทีใน Replit (Static HTML/CSS/JS project)
